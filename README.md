# Subscription Manager
## О проекте

**Subscription Manager** — это веб-приложение для управления подписками на цифровые сервисы.

Проект позволяет хранить все активные подписки в одном месте, получать напоминания о предстоящих списаниях, анализировать ежемесячные расходы и получать рекомендации по их оптимизации.

Проект разрабатывается в команде:

- **Соня** — Backend Developer
- **Насят** — Data Analyst

# 🎯 Цель проекта

Создать удобный сервис, который поможет пользователям:

- не забывать продлевать необходимые подписки;
- контролировать ежемесячные расходы;
- анализировать структуру затрат;
- получать рекомендации по экономии.

# 🚀 Основной функционал

### Управление аккаунтом

- регистрация пользователя;
- авторизация;
- личный кабинет.

### Управление подписками

Пользователь может:

- добавлять подписки;
- редактировать информацию;
- удалять подписки;
- просматривать список всех подписок.

Для каждой подписки хранится:

- название;
- категория;
- стоимость;
- дата следующего списания;
- период оплаты;
- статус активности.

---

# 🔔 Уведомления

Сервис автоматически напоминает пользователю **за один день до списания** средств по подписке.

---

# 📊 Аналитика

После добавления подписок пользователь получает аналитическую информацию:

- общая сумма расходов;
- расходы за текущий месяц;
- количество активных подписок;
- самая дорогая подписка;
- расходы по категориям;
- динамика расходов;
- список самых дорогих подписок;
- прогноз расходов на следующий месяц;
- рекомендации по сокращению затрат.

---

# 💡 Рекомендации

На основе данных о подписках приложение формирует рекомендации.

Например:

- какие подписки занимают большую часть бюджета;
- сколько можно сэкономить при отмене отдельных сервисов;
- какие крупные списания ожидаются в ближайшее время.
